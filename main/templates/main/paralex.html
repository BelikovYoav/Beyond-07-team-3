Lets Meet

There is a difficulty setting up meeting between people, teams and etc do to busy schedule that very between person to person.
As a result we want to present you our new platform that will help with the process of scheduling meetings.
Our platform is meant for everyone: students, work teams, friends who want to meet and more.

What We Offer

Set Meetings

Set Meetings


users can create a new meeting and set its character.
e.g. work meeting (preferred for student and work teams) or for Recreation and leisure (preferred for meeting with friends)

Upload Files

Upload Files


In work meetings files can be uploaded that will be linked to the event (at the end of the meeting or after it has ended)

Joined Meetings

Joined Meetings


Every user can create a meeting with other users.
He will define a range of dates that he prefers for the meeting to be set, how long the meeting will take and a list of wanted participants.

Our algorithm will return a range of dates that the meeting can be set, with all the participants involved,
or it will return that there is no such date in the relevant case.

Set Reminders

Set Reminders


Every user can set reminders for upcoming events

External Calendar

External Calendar


Every user can incorporate there external calendar into our calendar for e.g. google calendar
